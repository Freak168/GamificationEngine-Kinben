Reward Data Type

The super class of all rewards. Here are general information of the reward like to which organisation the reward belongs to or the goal to which it is associated. A Reward will be awarded in dependent of the goal for a group or one person after a person fulfilled a task. It can be chosen between one or more of the following rewards for a task: A permanent reward like a badge or achievement which can be obtained only once, or a volatile reward such as coins, points or a particular level which can be changed by getting for example more coins or be decreased by giving a bid for an offer in the marketplace.

Properties
name data type description
description string Gets the description of an achievement. This could contains for example the different tasks which have to be completed to get this achievement.
id number Gets the id of the reward which was generated by the creation.
name string Gets the name of an achievement which can describe the success in a short way and can be displayed in the application.
timeToLive number Gets the information about how long the reward can exist.
